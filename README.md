# agreste-master
Plan interactivo de agreste

## Sitio Web Básico

Este repositorio aloja un sitio web básico para el proyecto Agreste.

### Estructura del Proyecto

```
agreste-master/
├── index.html          # Página principal
├── css/
│   └── style.css      # Estilos CSS
├── js/
│   └── main.js        # Scripts JavaScript
├── images/            # Carpeta para imágenes
└── README.md          # Este archivo
```

### Cómo usar

1. **Personalizar el contenido**: Edita `index.html` para modificar el contenido de tu sitio web.

2. **Modificar estilos**: Edita `css/style.css` para cambiar la apariencia del sitio.

3. **Agregar funcionalidad**: Edita `js/main.js` para agregar interactividad con JavaScript.

4. **Agregar imágenes**: Coloca tus imágenes en la carpeta `images/` y referéncialas en el HTML.

### Ver el sitio web

Para ver el sitio web localmente:

1. Abre el archivo `index.html` directamente en tu navegador web.
2. O usa un servidor local como:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - VS Code: Usa la extensión "Live Server"

### GitHub Pages

Para publicar tu sitio en GitHub Pages:

1. Ve a la configuración del repositorio en GitHub
2. En la sección "Pages", selecciona la rama principal (main/master)
3. Guarda los cambios
4. Tu sitio estará disponible en: `https://guillermoahlig.github.io/agreste-master/`

### Personalización

Este es un sitio web básico de ejemplo. Puedes:
- Reemplazar todo el contenido con tus propios archivos HTML, CSS y JS
- Agregar más páginas HTML según necesites
- Integrar frameworks o librerías JavaScript de tu elección
- Agregar fuentes personalizadas, animaciones, etc.

¡Siéntete libre de personalizarlo completamente según tus necesidades!
